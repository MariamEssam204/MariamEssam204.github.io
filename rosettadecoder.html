Rosetta Decoder

Hieroglyphic detection and translation — from photo to story using YOLO, Gardiner codes, and generative AI.

Try it yourself

Upload a photo of hieroglyphic script and the app will translate it. Running on a Hugging Face Space — first load may take 20–30 seconds if the Space has been idle.

Open full demo on Hugging Face Spaces ↗

About the Project

  • Fine-tuned YOLO model detects individual hieroglyphic symbols in an uploaded image or live webcam feed
  • Each detected symbol is mapped to its Gardiner code via a structured code database
  • Exposed as MCP (Model Context Protocol) tools, so the same pipeline can be driven directly from Claude Desktop
  • Includes a full browsable reference table of supported Gardiner codes, grouped by category (deities, animals, plants, etc.)

Technical Implementation

🛠️ Detection & Translation Pipeline:
    • Detection: YOLO model fine-tuned on hieroglyphic symbols
    • Symbol mapping: Custom Gardiner code JSON database for sign descriptions and transliterations
    • Translation: API integration for dual-mode output (mystical narrative + academic translation)
    • Analytics: Matplotlib-generated frequency, confidence, and spatial heatmap charts
    • Tool exposure: Gradio's MCP server mode
    • Deployment: Gradio app on Hugging Face Spaces, embedded live on this site

Technologies Used

The implementation combines computer vision, generative AI, and agentic tool integration:

Python Ultralytics YOLO PyTorch Gradio MCP (Model Context Protocol) Matplotlib Hugging Face Spaces
