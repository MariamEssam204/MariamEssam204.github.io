About the Project

Fine-tuned a crab detection model for the MATE ROV Competition, supporting underwater object detection as part of the team's autonomous vision pipeline:

  • Fine-tuned YOLO11s on a custom-annotated, Roboflow-managed crab dataset
  • Multi-species detection in cluttered scenes with overlapping individuals
  • Deployed as a live, interactive Hugging Face Space

Technical Implementation

🛠️ Detection Pipeline:
    • Model: YOLO11s, fine-tuned from pretrained weights, 100 epochs
    • Dataset: Roboflow-annotated crab dataset (897 train / 80 val / 40 test images)
    • Deployment: Gradio app on Hugging Face Spaces, embedded live on this site

Technologies Used

The implementation combines a fine-tuned computer vision model with an interactive deployment pipeline:

Python PyTorch Ultralytics YOLO11 OpenCV Gradio Hugging Face Spaces Roboflow
