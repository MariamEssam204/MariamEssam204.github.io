Breast Cancer Detection System

Automate binary classification of breast tumors (benign vs. malignant) from mammograms to improve early detection

Deep Learning Classification of Benign vs Malignant Tumors

Case examples demonstrating correct classification of difficult borderline cases

Objective

The primary goal is to develop an AI-powered diagnostic assistant that:

  • Accurately differentiates between benign and malignant breast lesions.
  • Reduces false positives/negatives in early-stage detection.
  • Integrates seamlessly with existing medical imaging workflows.

Key Features

    • Dataset Integration: Used the INbreast and CBIS-DDSM datasets, integrating DICOM image metadata and Excel labels.
    • Metadata extraction: Handled metadata extraction and mapped labels and images for accurate training and evaluation.
    • Deep Learning Model: Implemented a convolutional neural network (CNN) for binary classification using PyTorch.
    • Preprocessing: Normalization, ROI extraction, and augmentation.
    • Data Analysis: Plotted BI-RADS distribution and explored tumor patterns for insights into dataset characteristics.
    • Performance metrics: Accuracy, precision, recall, and AUC-ROC.

Technologies Used

The implementation combines computer vision techniques and Data analysis for medical data:

Python Pytorch OpenCV CNN Features Extraction Data analysis
